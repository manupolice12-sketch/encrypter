# Encrypter
This is a project to encrypt entire folders it comes in 2 pieces 
This is very powerfull and compatible with Windows, Linux and MacOS
## CLI(Command Line Interface) 
CLI is just the same as GUI(Graphical user Interface) but it doesn't have a windows and you have to manually type the copy paste it 
## GUI
Here you select the folder using a pop up and type the password in the password entry box and this is manual in someways
